### Prerequisites

### Installation

1. Clone the repository
```bash
git clone https://github.com/ChittalaUday/cable_operator_app.git
```

2. Navigate to project directory
```bash

```

3. Install dependencies
```bash
flutter pub get
```

4. Run the app
```bash
flutter run
```
